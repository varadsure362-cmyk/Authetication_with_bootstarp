# Authentication App - Bootstrap 5

A simple authentication web application built using **HTML**, **Bootstrap 5**, and **Custom CSS**.

## Pages

| Page | File |
|------|------|
| Login | index.html |
| Register | register.html |
| Forgot Password | forgot-password.html |
| Reset Password | reset-password.html |
| Dashboard | dashboard.html |

## Technologies Used

- HTML5
- Bootstrap 5 (via CDN)
- Custom CSS (styles.css)
- Google Fonts (Poppins)

## Features

- Responsive design (works on mobile, tablet, desktop)
- Bootstrap card layout for all pages
- Bootstrap navbar on dashboard
- Custom color scheme
- Clean and simple UI

## How to Run

1. Clone or download this repository
2. Open `index.html` in your browser
3. Navigate through the pages using the links
